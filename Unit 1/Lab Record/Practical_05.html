HTML Example

HTML builds the structure of all web pages. Here's a simple example:

<!DOCTYPE html>
<html>
  <head>
    <title>My First Page</title>
  </head>
  <body>
    <h1>Hello, world!</h1>
  </body>
</html>

CSS Example

CSS (Cascading Style Sheets) is used to style HTML elements. Here's a simple example:

h1 {
  text-align: center;
  font-size: 24px;
}
p {
  line-height: 1.6;
  margin: 10px;
}
