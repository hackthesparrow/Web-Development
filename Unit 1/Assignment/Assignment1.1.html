HTML Attributes

1) What are Attributes?

Attributes are written inside the opening tag and provide additional information about an element — like name and value (name="value").

2) Example — <a> (href)

This attribute specifies the destination of a link.

Example: Open Wikipedia

Code:

<a href="https://www.wikipedia.org" target="_blank" title="Go to Wikipedia">Open Wikipedia</a>

3) Example — <img> (src, alt)

src defines the image path and alt is shown when the image fails to load.

Example:

Code:

<img src="Developer.jpeg" alt="Coding">

4) Some Other Common Attributes

  • class — assigns a group/class for CSS or JavaScript.
  • id — provides a unique identifier for each element.
  • title — shows a small tooltip when hovered over.
