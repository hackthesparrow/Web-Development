HTML Anchor Tag — Assignment

1) Purpose of the <a> Tag

The <a> tag (anchor tag) is used to create hyperlinks in HTML. These links connect one webpage to another, allowing navigation within or outside the website.

2) Syntax of the <a> Tag

<a href="URL" target="_blank" title="Description">Link Text</a>

Explanation of Each Part:

  • <a> — Opening anchor tag that defines the start of the hyperlink.
  • href="URL" — Specifies the destination address or webpage to open.
  • target="_blank" — (Optional) Opens the link in a new browser tab.
  • title="Description" — (Optional) Displays tooltip text when hovered over.
  • Link Text — The clickable text visible to the user.
  • </a> — Closing tag that ends the hyperlink.

3) Example

Rendered example: Visit Google

Code:

<a href="https://www.google.com" target="_blank" title="Go to Google Homepage">Visit Google</a>

4) Output Explanation

When you click the link, it opens the Google homepage in a new tab. The tooltip “Go to Google Homepage” appears when you hover over it.

5) Summary

The <a> tag in HTML is used to link web pages together, making navigation easy for users across documents and websites.
