HTML <img> Tag

1) Function of the <img> Tag

The <img> tag in HTML is used to display images on a webpage. It is an empty (self-closing) tag, meaning it doesn’t require a closing tag. The image can be from the same folder or from an external source (URL).

2) Common Attributes of the <img> Tag

  • src — Specifies the path or URL of the image file that will be displayed on the webpage.
  • alt — Provides alternative text that appears if the image fails to load. It also helps with accessibility for screen readers.

3) Example

Rendered example:

Code:

<img src="flower.jpg" alt="Beautiful Flower">

4) Importance of Attributes

  • src: Without this attribute, the browser doesn’t know which image to display.
  • alt: Helps users understand the content if the image is missing and improves SEO accessibility.
