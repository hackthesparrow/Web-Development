About This Page

This webpage is created using only <div> and <span> tags for layout.

It demonstrates how these basic elements can structure and style a webpage using CSS.

Why Use Div and Span?

Div is a block-level element — it creates sections or blocks of content.

Span is an inline element — used to style small parts of text or inline content.
